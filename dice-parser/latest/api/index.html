Dice#

class dice_parser.Roller(context: Optional[RollContext] = None)#

The main class responsible for parsing dice into an AST and evaluating that AST.

context#

The roll context associated with this roller.

Type:

RollContext

roll(expr: str | dice_parser.dice_ast.Node, stringifier: Optional[Stringifier] = None, allow_comments: bool = False, advantage: AdvType = AdvType.NONE) → RollResult#

Rolls the dice.

Parameters:

  • expr (Union[str, ast.Node]) – The dice expression

  • stringifier (Stringifier) – The stringifier to stringify the result. Defaults to MarkdownStringifier

  • allow_comments (bool) – Whether to parse for comments after the main roll expression (disables cache, can lead to possible performance issues)

  • advantage (AdvType) – If the roll should be made at (dis)advantage. Only applies if the leftmost node s 1d20

Returns:

The result of the roll

Return type:

RollResult

parse(expr: str, allow_comments: bool = False) → Expression#

Parses a dice expression into an AST.

Parameters:

  • expr (str) – The dice to roll

  • allow_comments (bool) – Whether to parse for comments after the main roll expression (disables cache, can lead to possible performance issues)

Returns:

An AST representing the dice expression.

Return type:

ast.Expression

class dice_parser.RollResult(the_ast: Node, the_roll: Expression, stringifier: Stringifier)#

Holds information about the result of a roll.

Instances should generally not be constructed manually.

int(x)

Return the integer result of the roll (rounded towards 0).

str(x)

Return the string result of the roll.

float(x)

Return the result of the roll without rounding.

ast#

The abstract syntax tree of the dice expression that was rolled

Type:

ast.Node

expr#

The Expression representation of the result of the roll.

Type:

Expression

comment#

If allow_comments was True and a comment was found, the comment. Otherwise, None.

Type:

Optional[str]

property total: int#

The integer result of the roll (rounded towards 0).

property result: str#

The string result of the roll. Equivalent to stringifier.stringify(self.expr).

property crit: CritType#

If the leftmost node was Xd20kh1, returns CritType.CRIT if the roll was a 20 and CritType.FAIL if the roll was a 1. Returns CritType.NONE otherwise.

class dice_parser.RollContext(max_rolls: int = 1000)#

Used to track information about rolls to ensure all rolls halt eventually.

max_rolls#

The maximum number of rolls allowed per context invocation

Type:

int

reset() → None#

Called at the start of each new roll.

Called each time a die is about to be rolled.

Parameters:

n (int) – The number of rolls about to be made

Raises:

Enumerations#

class dice_parser.AdvType(value)#

An enum representing the advantage a roll should be made at.

NONE: int#

Equivalent to 0. Represents no advantage.

ADV: int#

Equivalent to 1. Represents advantage.

DIS: int#

Equivalent to -1. Represents disadvantage.

class dice_parser.CritType(value)#

An enum representing the crit type of a roll.

NONE: int#

Equivalent to 0. Represents no crit.

CRIT: int#

Equivalent to 1. Represents a critical hit (a natural 20 on a d20).

FAIL: int#

Equivalent to -1. Represents a critical fail (a natural 1 on a d20).

class dice_parser.Stringifier#

An ABC for a string builder from a dice result.

Subclasses should implement all str_* methods to transform an Expression into a str.

str_binop(node: BinOp) → str#

str_set(node: Set) → str#
str_dice(node: Dice) → str#
str_die(node: Die) → str#
stringify(the_roll: Expression) → str#

Transforms a rolled expression into a string recursively, bottom-up.

stringify_node(node: Number) → str#

Called on each node that needs to be stringified.
