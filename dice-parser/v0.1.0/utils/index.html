dice_parser.utils.simplify_expr_annotations(expr: Number, ambig_inherit: Optional[Literal['left', 'right', None]] = None) → None#

Transforms an expression in place by simplifying the annotations using a bubble-up method.

Parameters:

  • expr (Number) – The expression to transform

  • ambig_inherit (Literal['left', 'right', None]) – Inheritance behaviour when encountering a child node with no annotation and a parent with ambiguous types. Can be None for no inherit, 'left' for leftmost, or 'right' for rightmost.
